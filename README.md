# 🎂 Birthday Bot

Телеграм-бот для поздравления с днём рождения. Работает в личных сообщениях и групповых чатах. Поздравления генерирует через GPT.

---

## Быстрый старт

### 1. Получи токены

| Что | Где взять |
|-----|-----------|
| Telegram Bot Token | [@BotFather](https://t.me/BotFather) → `/newbot` |
| OpenAI API Key | [platform.openai.com](https://platform.openai.com/api-keys) |

### 2. Настрой переменные окружения

```bash
cp .env.example .env
# Открой .env и заполни TELEGRAM_TOKEN и OPENAI_API_KEY
```

### 3. Запуск через Docker (рекомендуется)

```bash
docker-compose up -d
```

MongoDB поднимется автоматически вместе с ботом.

### 4. Или запуск вручную

```bash
# MongoDB должна быть запущена отдельно
pip install -r requirements.txt
python bot.py
```

## Использование

### Добавить людей — текстом

Напиши боту (или в группе) в любом из форматов:

```
@alex, Алексей, 15.03.1990
@masha, Мария, 22.07
@petya; Пётр; 01.01.1985
@vanya Иван 30.12.2000
```

- Год необязателен
- Разделители: `,` `;` или пробел

### Добавить людей — CSV-файлом

Прикрепи файл `.csv`:

```csv
username,name,birthday
@alex,Алексей,15.03.1990
@masha,Мария,22.07
```

### Команды

| Команда | Описание |
|---------|----------|
| `/start` | Запуск и регистрация чата |
| `/list` | Список всех добавленных людей |
| `/today` | Именинники сегодня (+ ручная отправка поздравлений) |
| `/delete @username` | Удалить человека |
| `/help` | Справка |

---

## Как работают поздравления

- Каждый день в **9:00** (настраивается в `.env`) бот проверяет все зарегистрированные чаты
- Если у кого-то сегодня день рождения — генерирует поздравление через GPT-4o-mini
- Одному человеку поздравление отправляется максимум **1 раз в год**

Пример поздравления:
> 🎉 В этот прекрасный день поздравляем @alex Алексея с днём рождения! Желаем тебе горы радости, моря здоровья и всего самого лучшего. Пусть этот год принесёт только хорошее! 🎂

## Переменные окружения

| Переменная | По умолчанию | Описание |
|-----------|-------------|----------|
| `TELEGRAM_TOKEN` | — | Токен бота (обязательно) |
| `MONGO_URI` | `mongodb://localhost:27017` | URI MongoDB |
| `MONGO_DB` | `birthday_bot` | Название базы данных |
| `OPENAI_API_KEY` | — | Ключ OpenAI (обязательно) |
| `CONGRATS_HOUR` | `9` | Час отправки поздравлений |
| `CONGRATS_MINUTE` | `0` | Минута отправки |
| `TIMEZONE` | `Europe/Moscow` | Временная зона |
